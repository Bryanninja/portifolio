Mais que um To-Do List

O Task Manager Pro é uma solução completa de produtividade pessoal. Diferente de listas comuns, ele implementa um conceito de "Well-being Productivity", integrando a gestão de tarefas por períodos do dia (Manhã, Tarde, Noite) com monitoramento de hidratação.

Destaques de Engenharia

Este projeto foi construído para demonstrar domínio sobre o ecossistema moderno do React. O foco não foi apenas "funcionar", mas ser escalável e performático.

  • React 19 & Vite: Utilizando as features mais recentes para máxima performance.
  • TanStack Query: Gerenciamento de estado assíncrono profissional, com cacheamento inteligente e re-fetching automático.
  • Robustez: Formulários controlados com React Hook Form e validação de esquemas via Zod.
  • UX/UI: Interface responsiva e acessível criada com Tailwind CSS e componentes do Radix UI (via Sonner para toasts).
  • Qualidade de Código: Pipeline de CI/CD configurado com Husky e Lint-staged para garantir padronização antes de cada commit.
