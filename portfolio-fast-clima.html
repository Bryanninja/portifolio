Fundamentos do React

O FastClima foi desenvolvido para demonstrar o domínio dos conceitos fundamentais do React.js sem a utilização de bibliotecas externas complexas. Aqui, a lógica é construída "na mão" para garantir o entendimento do ciclo de vida dos componentes.

Destaques Técnicos

O foco do projeto é o gerenciamento eficiente de estados e efeitos colaterais.

  • React Hooks: Uso intensivo de useState para controlar os dados da tela e useEffect para chamadas à API.
  • Integração de API: Consumo de dados assíncronos (OpenWeatherMap), tratando promessas e renderizando os dados dinamicamente.
  • Renderização Condicional: Controle lógico para exibir estados de carregamento, sucesso ou erro (feedback visual para cidade não encontrada).
